# SDK
{: #sdk}

要将 {{site.data.keyword.Bluemix}} Mobile Services SDK 添加到您的应用程序中，请选择要使用的 SDK，然后配置您的依赖关系管理器，以将 SDK 拉入您的应用程序中。
